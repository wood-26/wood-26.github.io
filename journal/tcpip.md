# The TCP/IP Model

## Application

Protocols in this layer include:

-WWW
	-HTTP
-SMS
-RCS
-Email
	-POP
	-IMAP
	-SMTP
-VOIP
-IRC
## Transport

Protocols in this layer include:

-UDP
-RP

## Internet

Protocols in this layer include:

-IP
-Routing
## Link (or Network Access)

Protocols in this layer include:
- Wifi
- Ethernet
- LTE
- Bluetooth
